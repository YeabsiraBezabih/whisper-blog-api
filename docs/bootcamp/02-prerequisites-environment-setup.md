# **2. Prerequisites & Environment Setup**

Before writing a single line of NestJS, you need a solid foundation and a working development environment. This section covers everything you need to install and configure.

## **Software to Install**

- Node.js (v20 LTS) — https://nodejs.org — the JavaScript runtime NestJS runs on

- npm (comes with Node) or pnpm / yarn for package management

- VS Code — https://code.visualstudio.com — your code editor

- Git — https://git-scm.com — version control

- Postman — https://www.postman.com/downloads — API testing client

- PostgreSQL (v15+) — https://www.postgresql.org/download — your database

- TablePlus or DBeaver — GUI tools to inspect your database visually

## **Essential VS Code Extensions**

- ESLint — real-time linting

- Prettier — Code Formatter — auto-format on save

- REST Client — test APIs right inside VS Code

- GitLens — enhanced Git capabilities

- TypeScript + Webpack Problem Matchers

- NestJS Snippets — productivity shortcuts for NestJS

- Prisma — syntax highlighting for Prisma schema files

- Thunder Client — lightweight Postman alternative inside VS Code

## **Install the NestJS CLI**

  npm install -g @nestjs/cli

Then scaffold your first project: nest new my-app
