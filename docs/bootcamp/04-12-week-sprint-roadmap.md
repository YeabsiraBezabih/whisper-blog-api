# **4. 12-Week Sprint Roadmap**

Each week is a focused sprint. Treat these like work sprints — start Monday, review Friday, reflect on the weekend. Aim for 2–3 hours of focused learning per day.

| **Week** | **Phase / Title** | **Topics Covered** | **Resources** | **Deliverable** |
|---|---|---|---|---|
| **Wk 1** | **TypeScript Bootcamp** | TS vs JS — why types matter<br>Primitive types, arrays, tuples, enums<br>Interfaces vs Type aliases<br>Functions: overloads, optional params<br>Classes, access modifiers<br>Generics basics<br>tsconfig.json explained | TypeScript Handbook (official)<br>Total TypeScript — Matt Pocock (free)<br>Execute Program — TypeScript track | Build a typed CLI task manager |
| **Wk 2** | **Node.js & HTTP Foundations** | Node.js event loop & non-blocking I/O<br>Built-in modules: fs, path, http<br>npm: package.json, scripts, devDeps<br>HTTP verbs, status codes, headers<br>Build a raw HTTP server (no framework)<br>Intro to Express (context only) | Node.js official docs (nodejs.org)<br>The Odin Project — NodeJS path<br>MDN HTTP overview | Raw Node HTTP server returning JSON |
| **Wk 3** | **NestJS Core Architecture** | NestJS philosophy & module system<br>Modules, Controllers, Services<br>Dependency Injection (DI) explained<br>Decorators: @Get @Post @Body @Param<br>DTOs (Data Transfer Objects)<br>First CRUD API: /users endpoints<br>Nest CLI commands | NestJS official docs (docs.nestjs.com)<br>NestJS Fundamentals — Udemy (Kamil)<br>NestJS crash course — YouTube Academind | Users CRUD API with in-memory store |
| **Wk 4** | **PostgreSQL & SQL Foundations** | Relational model: tables, keys, relations<br>Install PostgreSQL & psql CLI<br>Core SQL: SELECT WHERE ORDER JOIN<br>Aggregate functions: COUNT SUM AVG<br>Indexes: why and when to use them<br>pgAdmin / TablePlus GUI tour | PostgreSQL Tutorial (postgresqltutorial.com)<br>Mode SQL Tutorial (mode.com/sql-tutorial)<br>SQLZoo (interactive) | Design and query a school DB schema |
| **Wk 5** | **TypeORM + NestJS Integration** | TypeORM: entities, columns, decorators<br>@nestjs/typeorm setup & config<br>Repositories pattern<br>Migrations: generate & run<br>Relations: OneToOne OneToMany ManyToMany<br>Eager vs lazy loading | TypeORM official docs (typeorm.io)<br>NestJS DB chapter (docs.nestjs.com)<br>Ben Awad — TypeORM series (YouTube) | Connect Users API to real PostgreSQL DB |
| **Wk 6** | **Full CRUD App + Validation** | Build a Blog API (posts + users)<br>class-validator decorators<br>class-transformer: plainToInstance<br>Global ValidationPipe setup<br>Error response standardization<br>Pagination & filtering query params<br>Intro to Prisma (compare to TypeORM) | class-validator docs (GitHub)<br>Prisma docs (prisma.io/docs)<br>NestJS — Validation chapter | Full Blog API with validation |
| **Wk 7** | **Authentication with JWT** | Auth concepts: session vs token based<br>JWT: header.payload.signature deep dive<br>Passport.js: strategies overview<br>NestJS @nestjs/passport & @nestjs/jwt<br>Local strategy (username + password login)<br>JWT strategy (protect routes)<br>bcrypt for password hashing | NestJS Authentication chapter (official)<br>JWT.io — debugger & docs<br>Passport.js docs (passportjs.org) | Register / Login + protected routes |
| **Wk 8** | **Authorization & RBAC** | AuthGuard vs custom Guards<br>Roles: USER, ADMIN, MODERATOR<br>SetMetadata + Reflector pattern<br>Custom RolesGuard<br>Route-level vs global guards<br>Refresh token flow<br>Logout & token blacklisting | NestJS Authorization chapter (official)<br>OWASP Auth Cheat Sheet<br>Auth0 Blog: refresh token rotation | Admin-only routes in Blog API |
| **Wk 9** | **Advanced NestJS Patterns** | Exception Filters: custom error shapes<br>Interceptors: ResponseTransformInterceptor<br>Pipes: custom ParseUUIDPipe<br>Config module + .env validation (Joi/Zod)<br>File uploads with Multer<br>Swagger / OpenAPI with @nestjs/swagger | NestJS — Exception Filters chapter<br>NestJS — Interceptors chapter<br>@nestjs/swagger README (GitHub) | Auto-generated Swagger docs for Blog API |
| **Wk 10** | **Performance & Best Practices** | Caching with @nestjs/cache-manager (Redis)<br>Rate limiting: @nestjs/throttler<br>Database query optimization & indexes<br>Compression & response size<br>Logging with Winston / Pino<br>Health checks: @nestjs/terminus<br>Environment-based config (dev/prod/test) | NestJS Caching chapter (official)<br>Redis docs (redis.io)<br>Pino logger docs | Add caching + rate limiting to API |
| **Wk 11** | **Testing — Unit & Integration** | Jest fundamentals: describe it expect<br>Mocking: jest.fn() jest.spyOn()<br>Unit test: service layer (mock repos)<br>Unit test: controller layer<br>Supertest: HTTP integration tests<br>Test DB setup with in-memory SQLite / TestContainers<br>Coverage reports: --coverage flag | Jest official docs (jestjs.io)<br>NestJS Testing chapter (official)<br>Supertest README (GitHub) | 80%+ coverage on Blog API |
| **Wk 12** | **Postman, E2E & Deployment** | Postman: collections, environments, variables<br>Postman: pre-request scripts & test scripts<br>Newman: run collections from CLI<br>NestJS E2E test suite<br>Docker basics: Dockerfile for NestJS<br>Docker Compose: app + PostgreSQL<br>Deploy to Railway / Render (free tier) | Postman Learning Center (learning.postman.com)<br>Newman docs (GitHub)<br>Docker Getting Started (docs.docker.com) | Blog API live & tested end-to-end |
